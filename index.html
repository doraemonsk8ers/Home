Welcome to Home of Doraemon.Sk8ers.

Recently, our team have read a recent blog entry by Piotr Bania

The article is about a critical vulnerability which has been found in TP-LINK routers and few other router devices that allows attackers to download Rom-0 through a simple HTTP request.

Piotr Bania's article is about patching this vulnerability, whereas we decide to decrypt this 'rom-0' file.

What is so special about this Rom-0 file is that ADSL login/password combination, WIFI password and basically all of your configuration data are stored within this file.

However, there isn't any reliable tool that will decrypt the rom-0 of our router.

So we decided to write the tool, Rom-0 Decoder in C# and share the tool with everyone.


We hope that someone will find it useful.
